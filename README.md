E-commerce Campaign Performance Analysis
Project Overview

This project analyzes e-commerce advertising campaign performance using Excel and Power BI. The objective is to evaluate campaign efficiency and identify optimization opportunities using key advertising metrics such as Spend, Sales, CTR, and ROAS.

Business Questions
Question 1 (Excel)

Calculate packaging dimensions and shipping volume using product measurements.
Objective: Transform raw product data and compute packaging metrics required for logistics.

Question 2 (Excel)

Perform sales and inventory planning calculations to determine stock requirements based on sales data.

Question 3 (Power BI)

Build a PPC campaign performance dashboard to analyze advertising metrics including:

Total Spend

Total Sales

CTR (Click Through Rate)

ROAS (Return on Ad Spend)
